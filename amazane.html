AmaZane

Mock-Amazon E-commerce application named after Bootcamp instructor.

About the App!

For this project, we wanted to create our very own e-commerce store that sold the brands that we love and enjoy. My responsibilities consisted of the front-end design and all UI components, as well as the creation of our database and the rendering of all the items via the use of Handlebars.

Technologies/Concepts

  • Client Side:
    • HTML5/CSS3
    • Materialize
    • JavaScript
    • Handlebars
    • Axios
  • Server Side:
    • Node.js
    • Express
    • MongoDB
