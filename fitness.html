Fitness/Workout Tracker

Track your workouts with this easy-to-use Fitness Tracker.

About the App!

For this assignment, we were asked to make an application that a user could use to create and track daily workouts. Additionally, a user should be able to log multiple workouts within a single day, track their personal info (name, weight, sets, reps, etc.), as well as track the distance they travel during a cardio exercise.

Technologies/Concepts

  • Client Side:
    • HTML5/CSS3
    • Semantic UI
  • Server Side:
    • Node.js
    • Express
    • Mongoose
