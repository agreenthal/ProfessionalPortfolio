Online/Offline Budget Tracker

Track your finances with this easy-to-use Budget Tracker.

About the App!

Every person wishes that they could have a simple, easy to use app that enables them to keep track of their financial situation. Something without the extra, unneccessary features. With this project, we were tasked with creating a design and applying functionality for an application just like that, one that users can use by logging expenses and deposits. The main focus of the project was learning how to create an application that can function both online and offline.
